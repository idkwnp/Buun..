# Buuñ

ADVERTENCIA: la musica/efectos de sonidos pueden ser un poco rompe timpanos, asi que es recomendable bajar el volumen antes de jugar

Este juego fue echo en mi colegio y decidi publicarlo aca
(este juego fue echo con Gemini porque la consigna era usarlo para crear un juego)

¿De que es el juego? El juego es un shooter estilo Doom

Creditos de los creadores de Die of Death por la musica/efectos de sonido

por ahora no es compatible con telefono

Controles

-"Q" para activar el escudo

-"SHIFT" para dashear

-Click izquierdo para disparar

-click derecho para puntar (te hace mas lento)

-"E" para usar uno de tus botis (si tienes uno)

-"WASD" para moverse

-"Esc" para abrir el menu de pausa

-"F3" para tercera persona
